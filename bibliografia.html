Introducción al Desarrollo Web

Si está aprendiendo sobre desarrollo web, encontrará términos como HTML, CSS y JavaScript. A menudo, estos se denominan los componentes básicos de la Web.Estas tres herramientas dominan el desarrollo web. Cada biblioteca o herramienta parece estar centrada en HTML, CSS y JS. Entonces, si desea convertirse en desarrollador web, debe aprenderlos bien. También descubrirá que los sitios web se crean principalmente a partir de estos tres idiomas. Pero probablemente se esté preguntando qué es cada uno y para qué se usa realmente. ¿Qué hace que estos idiomas sean tan especiales e importantes? HTML proporciona la estructura básica de los sitios, que es mejorada y modificada por otras tecnologías como CSS y JavaScript. CSS se utiliza para controlar la presentación, el formato y el diseño. JavaScript se utiliza para controlar el comportamiento de diferentes elementos. Internet es simplemente una red de computadoras que se comunican entre sí para enviar y recibir datos (información). Cada una de estas computadoras en Internet se puede distinguir y ubicar por un número único llamado dirección IP. Una dirección IP se parece a esto: 168.212.226.204 La Web es un subconjunto de Internet. Como cualquier otra red informática que existe, la Web se compone de dos componentes principales: el cliente del navegador web y el servidor web. El cliente solicita los datos y el servidor comparte o sirve sus datos. Para lograr esto, las dos partes deben establecer un acuerdo. Ese acuerdo se denomina Interfaz de programación de aplicaciones o, en resumen, API. Pero estos datos deben organizarse y formatearse en una forma que sea comprensible para los usuarios finales que tienen una amplia gama de experiencias y habilidades técnicas. Aquí es donde entran en juego HTML, CSS, JavaScript y todo el concepto de desarrollo web. Juntos, usamos estos tres lenguajes para formatear, diseñar y programar páginas web. Y cuando enlaza algunas páginas web con hipervínculos, junto con todos sus activos, como imágenes, videos, etc., que se encuentran en la computadora del servidor, se procesa en un sitio web. Esta representación generalmente ocurre en la interfaz, donde los usuarios pueden ver lo que se muestra e interactuar con él. Por otro lado, los datos, especialmente la información confidencial como las contraseñas, se almacenan y suministran desde la parte trasera del sitio web. Esta es la parte de un sitio web que existe solo en la computadora del servidor y no se muestra en el navegador frontal. Allí, el usuario no puede ver o acceder fácilmente a esa información.
